Reflection

In this project I created a webpage that displays playing cards using HTML and CSS. I used Flexbox to align the cards and control spacing between them. Each card has three sections called left, middle, and right which helped me practice alignment. This project helped me understand how Flexbox works and how HTML elements can be organized to build a layout.
